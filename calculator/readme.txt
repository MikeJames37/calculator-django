Проект по созданию калькуляторов раскроя металла и металло-консктрукций на Django.

Функционал калькуляторов:
Калькулятор раскроя листового металла (оптимизация размещения деталей на листе).
Расчёт длины развёртки гнутого профиля (по радиусу, углу, толщине металла).
Расчёт веса металлопроката (по размерам и материалу).
Оптимизация раскроя труб/профилей (минимизация отходов).
Расчёт себестоимости раскроя (учитывая стоимость материала и резки).

Функционал каждого калькулятора:
Ввод параметров через формы.
Визуализация результата (график, таблица).
Сохранение расчёта в PDF/Excel (для зарегистрированных).

Установка (pip install -r requirements.txt)

Запуск (python manage.py runserver)

Правила написания коммитов:

Типы коммитов (основные):
    feat – новая функциональность.
    fix – исправление бага.
    docs – изменения в документации.
    style – правки форматирования (пробелы, запятые), без изменений кода.
    refactor – рефакторинг кода без изменения поведения.
    perf – изменения, улучшающие производительность.
    test – добавление/правка тестов.
    chore – вспомогательные изменения (конфиги, зависимости).

Примеры коммитов: fix(login): handle empty password case
