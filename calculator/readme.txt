Проект по созданию калькуляторов раскроя металла и металло-консктрукций на Django.

Установка (pip install -r requirements.txt)

Запуск (python manage.py runserver)

Правила написания коммитов:

Типы коммитов (основные):
    feat – новая функциональность.
    fix – исправление бага.
    docs – изменения в документации.
    style – правки форматирования (пробелы, запятые), без изменений кода.
    refactor – рефакторинг кода без изменения поведения.
    perf – изменения, улучшающие производительность.
    test – добавление/правка тестов.
    chore – вспомогательные изменения (конфиги, зависимости).

Примеры коммитов: fix(login): handle empty password case
